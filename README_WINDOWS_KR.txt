============================================================
  SpatchEx UAT 도구  —  Windows 사용 가이드
============================================================

이 도구는 S-Patch EX 앱의 장기 ECG 테스트 중
증상(Symptom)을 자동으로 입력해주는 도구입니다.
개발 지식 없이 누구나 사용할 수 있습니다.


------------------------------------------------------------
  시작 전 확인 — ZIP 차단 해제
------------------------------------------------------------

이메일이나 공유 폴더에서 받은 ZIP 파일은
Windows 보안 정책에 의해 차단될 수 있습니다.

해제 방법:
  1) ZIP 파일을 마우스 오른쪽 버튼으로 클릭
  2) [속성] 선택
  3) 하단에 [차단 해제] 체크박스가 보이면 체크 후 [확인]
  4) ZIP 압축 해제 후 사용

※ 이 단계를 건너뛰면 install.bat 실행 시 오류가 발생할 수 있습니다.


------------------------------------------------------------
  1. 최초 1회 설치
------------------------------------------------------------

  1) Android 스마트폰을 USB 케이블로 PC에 연결합니다.

  2) 스마트폰에서 USB 디버깅을 활성화합니다.
       설정 → 휴대전화 정보 → 소프트웨어 정보
       → [빌드 번호]를 7번 빠르게 탭
       설정 → 개발자 옵션 → USB 디버깅 ON

  3) 스마트폰에 "USB 디버깅을 허용하시겠습니까?" 팝업이 뜨면
     [허용]을 탭합니다.

  4) install.bat 을 더블클릭합니다.
     Python, Node.js, ADB, Appium, Python 패키지가 자동으로 설치가 진행됩니다.

  ※ 처음 설치 시 5~10분 정도 걸릴 수 있습니다.
  ※ CMD 창이 멈춘 것처럼 보이면 Enter 를 한 번 눌러보세요.


------------------------------------------------------------
  2. 테스트 실행
------------------------------------------------------------

  1) S-Patch EX 앱을 켜고 ECG 측정이 진행 중인지 확인합니다.

  2) run.bat 을 더블클릭합니다.
     잠시 후 브라우저가 자동으로 열립니다.
     열리지 않으면 직접 접속하세요:  http://127.0.0.1:5001

  3) 브라우저에서 아래 항목을 설정합니다:
     - Device       : 연결된 기기를 선택합니다
     - Test Name    : 테스트 이름을 입력합니다
     - Tester Name  : 테스터 이름을 입력합니다
     - Team Hub URL : 원격 모니터링이 필요하면 입력합니다 (선택)
     - Duration     : 테스트 총 시간을 설정합니다
     - Interval     : 증상 입력 간격을 설정합니다
     - Symptoms     : 자동 입력할 증상을 선택합니다

  4) [Start Test] 버튼을 클릭하면 자동 테스트가 시작됩니다.

  5) 테스트가 끝날 때까지 run.bat 창을 닫지 마세요.


------------------------------------------------------------
  3. 테스트 중지
------------------------------------------------------------

  STOP.bat 을 더블클릭하거나
  브라우저에서 [Stop Test] 버튼을 클릭하세요.


------------------------------------------------------------
  4. USB / WiFi 연결 안내
------------------------------------------------------------

  처음 사용할 때는 USB 연결이 필요합니다.
  이후 동일 PC/Mac에서 WiFi ADB 설정이 완료된 기기는 무선으로도 사용할 수 있습니다.

  이전에 WiFi ADB 설정이 완료된 기기는
  환경 변수 SPATCH_DEVICE_IP 를 설정하면
  run.bat 실행 시 자동으로 WiFi 연결을 시도합니다.

  SPATCH_DEVICE_IP 를 설정하는 방법:
    시스템 속성 → 환경 변수 → 새로 만들기
    변수 이름: SPATCH_DEVICE_IP
    변수 값:   기기 IP 주소 (예: 192.168.0.41)

  ※ WiFi 연결이 실패하더라도 프로그램은 정상 실행됩니다.
  ※ 별도의 WiFi 전용 스크립트는 필요 없습니다.


------------------------------------------------------------
  5. 자주 발생하는 문제
------------------------------------------------------------

  기기가 인식되지 않는 경우:
    - USB 케이블을 뽑았다가 다시 꽂아보세요.
    - USB 디버깅이 켜져 있는지 확인하세요.
    - 스마트폰 화면에 "허용" 팝업이 있으면 탭하세요.

  install.bat 실행 오류:
    - install.bat 을 다시 실행하면 대부분 해결됩니다.
    - 실행 시 "Windows가 PC를 보호했습니다" 창이 뜨면
      [추가 정보] → [실행] 을 클릭하세요.

  스크립트가 멈춘 것처럼 보일 때:
    - Enter 키를 한두 번 눌러보세요.

  브라우저가 열리지 않는 경우:
    - http://127.0.0.1:5001 을 직접 입력하세요.

  테스트가 계속 실패하는 경우:
    - 브라우저의 [Failures] 탭에서 스크린샷과 오류를 확인하세요.


------------------------------------------------------------
  6. 팀 대시보드 (선택 사항)
------------------------------------------------------------

  README_TEAM_DASHBOARD_KR.txt 를 참고하세요.
