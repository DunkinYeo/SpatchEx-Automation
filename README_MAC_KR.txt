============================================================
  SpatchEx Automation (Mac)
============================================================

이 프로그램은 S-Patch Ex 앱 테스트 중
증상(Symptom)을 일정 주기로 자동 입력하기 위한 테스트 자동화 도구입니다.

Mac 환경에서 S-Patch Ex 앱 테스트를 수행할 때
증상 추가를 자동으로 반복 실행하여 장시간 테스트를 도와줍니다.


------------------------------------------------------------
  1. 설치 방법
------------------------------------------------------------

  install.command 실행

  Finder에서 install.command 파일을 더블 클릭하여 실행합니다.

  설치 과정에서 아래 항목들이 자동으로 준비됩니다:

    - Homebrew 확인 및 설치
    - Python 3.10+
    - Node.js / npm
    - Android platform-tools (adb)
    - Appium
    - Appium UiAutomator2 driver
    - Python 가상환경 (.venv)
    - Python 패키지 설치

  처음 실행 시 설치 과정이 3~5분 정도 걸릴 수 있습니다.


------------------------------------------------------------
  2. 프로그램 실행
------------------------------------------------------------

  설치가 완료되면 run.command 를 실행합니다.

  Finder에서 run.command 를 더블 클릭하면:

    - Python 환경 활성화
    - adb (Android SDK) 확인
    - Appium 서버 실행
    - Web UI 실행

  브라우저가 자동으로 열립니다.

  주소:
    http://127.0.0.1:5001


------------------------------------------------------------
  3. 테스트 시작 방법
------------------------------------------------------------

  웹 화면에서 아래 항목을 설정 후 테스트를 시작합니다:

    - Device (연결된 Android 기기)
    - Test Name
    - Tester Name
    - Test Duration
    - Symptom Interval
    - Symptoms (자동 입력할 증상)

  Start Test 버튼을 누르면 자동 테스트가 시작됩니다.


------------------------------------------------------------
  4. 필수 조건
------------------------------------------------------------

  - Android 기기가 USB로 Mac에 연결되어 있어야 합니다.
  - Android 기기에서 "USB Debugging" 이 활성화되어 있어야 합니다.
  - 인터넷 연결이 필요합니다 (설치 시).


------------------------------------------------------------
  5. 참고 사항
------------------------------------------------------------

  - 처음 설치 시 일부 과정에서 사용자 확인이 필요할 수 있습니다.
  - Homebrew 설치 시 Mac 비밀번호 입력이 필요할 수 있습니다.
  - Appium 및 드라이버 설치에는 몇 분 정도 시간이 걸릴 수 있습니다.


------------------------------------------------------------
  6. 문제 해결
------------------------------------------------------------

  설치가 정상적으로 진행되지 않을 경우:

  1) 인터넷 연결 확인

  2) 아래 명령어 실행 후 다시 시도:

       chmod +x install.command
       chmod +x run.command

  3) install.command 다시 실행
